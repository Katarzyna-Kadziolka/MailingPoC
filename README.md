# MailingPoC

Proof of concept for mailing.
### Features
- [Handlebars.Net](https://github.com/Handlebars-Net/Handlebars.Net) to handle dynamic HTML templates
- Two ways of sending emails:
    - [LocalStack](https://www.localstack.cloud/) with [AWS SES](https://aws.amazon.com/ses/)
    - [Mailhog](https://github.com/mailhog/MailHog)

**See more on my [blog](https://chihuahuacoder.com/) :)**
- [Dynamic emails with Handlebars.Net](https://chihuahuacoder.com/dynamic-emails-with-handlebarsnet)
- How to set up AWS SES with LocalStack (soon)
